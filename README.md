## Current what the playground has:

### Mortgage Calculator

A simple Java console application that:

* Takes a **loan principal**
* Takes an **annual interest rate**
* Takes the **loan period in years**

The program then:

* Calculates the **monthly mortgage payment**
* Displays a **payment schedule**
* Formats output as **currency**

Concepts practiced in this project:

* Java classes and objects
* Constructors
* Input validation
* Loops
* reducing coupling
* abstraction
* encapsulation
* Mathematical calculations
* Currency formatting using `NumberFormat`
* Basic JavaDoc documentation

## Project Structure

```
src/
 └─ com.medhat
     ├─ Main.java
     └─ Mortgage.java
```

**Main.java**

Program entry point.
Initializes input and starts the mortgage calculator.

**Mortgage.java**

Contains the core logic for:

* Reading user input
* Validating values
* Calculating monthly payments
* Printing the payment schedule
